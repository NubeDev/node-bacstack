getEnumName(object, Number)

Source:
  • enum.js, line 20
Turn an enum into a string suitable for debugging.
Example
const s = bacnet.enum.getEnumName(
  bacnet.enum.PropertyIdentifier,
  bacnet.enum.PropertyIdentifier.PRESENT_VALUE
);
console.log(s); // "PRESENT_VALUE(85)"
Parameters:
Name Type Description
object group Enum group, e.g. bacnet.enum.ConfirmedServiceChoice.
Number value Enum value, e.g. 1. Note that this *must* be an integer value, so you may need to use parseInt(). Non-integer values will result in an exception.
